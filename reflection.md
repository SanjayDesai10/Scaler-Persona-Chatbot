# Personal Reflection & Learnings — Scaler Persona Chatbot

Creating this persona chatbot allowed me to practice the concepts in lecture in a useful and tangible way. I treated every system prompt like a mini product description. The identity set the tone of the persona, few-shot examples gave us insight into model behavior better than just some words like "helpful," and rules about output formatting (like reply length, trailing questions, no bulleted lists) kept it out of some bland "assistant" persona mode. The usage of 3 distinct prompts across 1 API enforced discipline.

I was constantly reminded of the lesson about GIGO. I had early drafts of personas that just said "you are X, be helpful," and no matter what model I used, the responses were not good. Detailed research to find actual roles and plausible phrases with scenario specific examples along with strict formatting requirements dramatically improved the output. Weak prompt = weak persona, a stronger prompt doesn't guarantee success, but the failures are easier to trace. The feedback loop of GIGO was always present, but more as something to take responsibility for rather than placing blame.

On the product side, decoupling persona switching from message sending cleaned up the state: the backend thread resets on persona switch, and the frontend needed to clear messages simultaneously to stop it from displaying a reply in an old persona's history. While streaming a response significantly improved the perceived latency, it was another place where I had errors to be handled.

Going forward I would build out some lightweight regression fixtures for the project (with fixed user prompts for each persona and manual reviews of snapshots), implement better structured logging for API errors, and potentially rate limit the chat route. This project showed that prompt engineering is just more product engineering: ship it, watch it fail, tighten the screws and do it again. The product around the model (errors, reset semantics, suggested chips, etc.) is almost as important as the system prompt.
